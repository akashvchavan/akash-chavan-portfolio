# Akash Chavan Portfolio

React + TypeScript portfolio built with Vite.

## Run locally

1. Install Node.js.
2. Open a terminal in this folder.
3. Run `npm install`.
4. Run `npm run dev`.
5. Open the local URL shown by Vite.

## Before publishing

Open `src/main.tsx` and replace:
- `YOUR_EMAIL_HERE`
- `YOUR_LINKEDIN_URL_HERE`
- `YOUR_GITHUB_URL_HERE`

## Production build

Run `npm run build`.
